# QuizMaster — Online Quiz Management System

A full-stack web application for managing and attempting online quizzes.

## Tech Stack
- Backend: Python Flask
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## Features
- Admin panel to create and manage quizzes
- Student interface to attempt quizzes with live scoring
- User authentication and role-based access
- Result analytics dashboard
